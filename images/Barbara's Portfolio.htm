My UX Skills

Wireframing

Create the best possible blueprint for how something should work. Ensuring all elements and interface components for interactions are included. The tools used for this are paper and pencil, Figma, and InVision. The skills I use are curiosity, to figure how it should work for the user.As well as empathy and research.

User Empathy

When it comes to user empathy, i want to learn how everything makes the user feel, from the moment they begin their process to the end. This means speaking with both people who have had great experiences, and listening to the not so great ones for improvements. The skills/tools i would user are research, communication, empathy, google drive/miro to note my findings.

Communication & Collaboration

Effectively communicating product ideas to improve products and systems. Work collaboratively with other teams. Provide and receive feedback, explore different solutions, and incorporate the expertise of everyone in a project to create the best possible product.
